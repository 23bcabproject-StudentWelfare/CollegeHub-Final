🚀 AI Career Dashboard & CV Builder

A modern AI-powered career dashboard and resume builder designed to help students and job seekers track skills, build professional resumes, and prepare for career growth.

The platform provides an interactive dashboard with analytics, career insights, and a guided CV creation system that allows users to generate a professional resume step-by-step.
